# whirlpool-client-gui

Desktop GUI for [Whirlpool](https://github.com/Samourai-Wallet/Whirlpool) by Samourai-Wallet.


# Installation

### Requirements
- npm (sudo apt install npm)
- yarn (sudo npm install -g yarn)

### Install
```
cd whirlpool-gui
yarn
```

# Usage
```
cd whirlpool-gui
yarn dev
```
